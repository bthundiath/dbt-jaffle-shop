# dbt Learning Project

Learning dbt Core with PostgreSQL following the dbt Fundamentals course.

## Stack
- dbt Core 1.11
- PostgreSQL (local)
- Python 3.x

## Project
- Jaffle Shop sample project
- Models, tests, seeds, and documentation

## Commands
```bash
dbt seed      # load raw data
dbt run       # build models
dbt test      # run data tests
dbt docs generate && dbt docs serve  # view documentation
```

### Using the starter project

Try running the following commands:
- dbt run
- dbt test
